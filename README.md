# IshowspeedBot

## basically:
A discord bot for ishowspeed fans

## Important feautures:
- Notify every time ishowspeed goes live on youtube
- When asked it'll give the latest news on ishowspeed 

## Commands: 
- /bark
>Ishowspeed joins voice call and barks
- /ping
>Replies with Pong!
- /reload
>Reloads a command (guild only)

## Setup
1. Add your bot token to `.env` as `BOT_TOKEN=...`
2. Install dependencies: `npm install`
3. Register slash commands: `npm run deploy`
4. Start bot: `npm start`
